void AudioRecorder::startRecording ( )

Starts recording audio.

If already recording, this function returns early. Otherwise, it initializes the audio stream and spawns the recording thread.

◆ stopRecording()

void AudioRecorder::stopRecording ( )

Stops recording audio.

If not currently recording, this function throws a runtime error. Otherwise, it stops and closes the audio stream, and joins the recording thread.
